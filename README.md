# uni-music

### 1、**项目介绍**：
- uni-app、uview-ui、小程序集于一身的实战，很好的一个练手项目。功能有：首页推荐、音乐列表页展示、音乐详情页、音乐播放器、推荐音乐、个人中心、登入，注册，粉丝，关注，音乐搜索、mv推荐，mv播放，等实现。
